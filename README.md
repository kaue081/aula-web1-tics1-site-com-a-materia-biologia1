## ⚙️ Pré-requisitos

Antes de clonar e rodar o projeto, certifique-se de ter as seguintes ferramentas instaladas na sua máquina:

* [Git](https://git-scm.com/) (para clonar o repositório)
* [Node.js](https://nodejs.org/) (versão LTS recomendada)
* [VS Code](https://code.visualstudio.com/) (ou seu editor de código preferido)

---

## 🚀 Instalação e Configuração

Quando você faz o clone deste repositório, a pasta `node_modules` não é incluída no download. Isso é um comportamento padrão (configurado no `.gitignore`) para evitar que arquivos pesados e gerados automaticamente lotem o repositório.

Para baixar e instalar todas as dependências necessárias, abra o terminal na pasta raiz do projeto e execute:

```bash
npm install
```
